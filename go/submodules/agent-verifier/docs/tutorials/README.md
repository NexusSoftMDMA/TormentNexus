# Agent Verifier Tutorials

Step-by-step guides to help you get the most out of Agent Verifier.

## Tutorials

| # | Tutorial | Time | What you'll learn |
|---|----------|------|-------------------|
| 1 | [Installation](01-installation.md) | 5 min | Install Agent Verifier with any method |
| 2 | [First Verification](02-first-verification.md) | 10 min | Run your first verification and save reports |

## Prerequisites

- **Node.js** v18 or later (for the skills CLI)
- **A supported coding agent** — Claude Code, Roo Code, Cursor, Windsurf, or [others](https://github.com/vercel-labs/skills#supported-agents)

## Quick Reference

| Task | Command |
|------|---------|
| Full verification | `"verify agent"` |
| Security checks only | `"verify agent security"` |
| Pattern checks only | `"verify agent patterns"` |
| Quality checks only | `"verify agent quality"` |
| Language checks only | `"verify agent language"` |
