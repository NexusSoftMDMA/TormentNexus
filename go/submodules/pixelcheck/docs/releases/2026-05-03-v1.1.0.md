## Internal-only signature change

`OpenedExtractor.readMetrics: () => StagehandMetricsSnapshot` →
`() => Promise<StagehandMetricsSnapshot>` (forced by Stagehand v3
making `stagehand.metrics` async). `OpenedExtractor` is internal —
no public-API consumer impact. Mock callers using `() => ({...})`
continue to work because `await` on a sync return resolves to the
value unchanged.

## Rollback plan

If v1.1.0 surfaces an unforeseen regression:

```bash
# Code rollback (npm dist-tag): point `latest` back at v1.0.1
npm dist-tag add pixelcheck@1.0.1 latest

# Or fully unpublish v1.1.0 (within 72h grace window only)
npm unpublish pixelcheck@1.1.0
```

Database rollback: not needed — no schema migrations in v1.1.0.
