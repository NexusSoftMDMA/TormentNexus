## Summary

Trivial one-line documentation patch. The v1.1.3 CHANGELOG entry contained one residual reference to the prior dogfood product's filename inside a "before / after" rename description; that line has been rephrased to be fully generic. Runtime code, bundled assets, all other documentation, and the npm tarball file list are otherwise unchanged from v1.1.3.

This release exists solely to ensure no shipped artifact contains any reference to the maintainer's prior dogfood product, as a strict reading of the cleanup goal that produced v1.1.3.
