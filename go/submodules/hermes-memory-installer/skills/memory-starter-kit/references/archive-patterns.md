# Archive Patterns

## Naming Conventions
- People: `archives/people/firstname-lastname/profile.md`
- Projects: `archives/projects/project-name/overview.md`
- Knowledge: `archives/knowledge/topic-name.md`

## Tag Conventions
- People: `friend`, `colleague`, `family`, `professional`
- Projects: `active`, `completed`, `on-hold`
- Knowledge: `technical`, `personal`, `reference`, `guide`

## Cross-Referencing
Use `[[wikilinks]]` to reference other archives:
- `[[people/alice]]` — Link to a person
- `[[projects/my-app]]` — Link to a project
