# Writing Guide for Archives

## General Principles
- **Be specific**: Include dates, names, numbers where possible
- **Be concise**: Each entry should be scannable in under 30 seconds
- **Use tags**: Tags enable cross-referencing and search

## Person Archives
- Update `last_updated` after every meaningful interaction
- Add timeline entries for key events
- Note relationship dynamics objectively

## Project Archives
- Track decision rationale, not just the decision
- Link to related people and knowledge entries
- Mark milestones with concrete dates

## Knowledge Archives
- Cite sources clearly
- Link to related projects or people
- Update when understanding evolves
