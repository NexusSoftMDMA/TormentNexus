Connection Examples

Connect AI tools to this MCP server through Arc Relay.

arc-sync CLI (Recommended)

The easiest way to connect. Handles config files, auth headers, and per-project management automatically. 

arc-sync add {{.Server.Name}}
First-time setup & install 
# One-line install & setup (Linux/macOS)
curl -fsSL {{.BaseURL}}/install.sh | bash

# Or install manually, then init with browser-based auth
arc-sync init {{.BaseURL}}
Claude Code (.mcp.json) - manual setup

Add to .mcp.json in your project root:

{
  "mcpServers": {
    "{{.Server.Name}}": {
      "type": "http",
      "url": "{{.BaseURL}}/mcp/{{.Server.Name}}",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
Claude Desktop OAuth

Add this server to Claude Desktop as a custom connector. Authentication is handled automatically via OAuth.

{{.BaseURL}}/mcp/{{.Server.Name}}
Setup steps
  1. In Claude Desktop, click your profile icon > Settings
  2. Go to Connectors > Add custom connector
  3. Paste the URL above and click Add
  4. A browser window will open - log in and click Approve
  5. Return to Claude Desktop - the server should show as connected
