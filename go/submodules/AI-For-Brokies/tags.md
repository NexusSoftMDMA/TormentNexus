# Tags

This document provides descriptions for all tags used to categorize tools in this repository.

| Tag             | Description                                                              |
|-----------------|--------------------------------------------------------------------------|
| `Model`         | Tools that are AI model applications or platforms                        |
| `Desktop-App`   | Standalone desktop applications                                          |
| `IDE`           | Integrated Development Environments with built-in AI capabilities        |
| `IDE-Extension` | Extensions/plugins that add AI features to existing IDEs                 |
| `Terminal-App`  | Command-line interface tools and applications                            |
| `Open-Source`   | Tools with publicly available source code                                |
| `Student-Deal`  | Tools offering special pricing, free tiers, or benefits for students     |
| `Api-Optional`  | Tools that work without an API key but can use one for enhanced features |
| `Api-Required`  | Tools that require an API key to function                                |
