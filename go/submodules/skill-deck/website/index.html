◇ the problem

Fifty skills. Six agents. Zero overview.

Every coding agent invented its own skill format. Skill Deck reads all of them into one place.

▸ Without Skill Deck

Claude Code drops SKILL.md. Cursor has .mdc. Copilot has .prompt.md. Codex, Windsurf, Gemini, Cline, Roo, Continue, Aider — each with a different folder, a different layout, a different convention.

You install fifty skills across six agents and lose track of what you have, what you actually use, and which ones are stale. You hunt through dotfiles to remember a slash command. You re-read a SKILL.md you wrote three months ago because no UI lets you skim them all.

▸ With Skill Deck

Press Ctrl+Shift+K. A panel slides in. Every skill, command, hook, rule, prompt, and workflow you have across every coding agent — in one searchable list.

Click one, see the body. Star the good ones. Snapshot before you let an agent overwrite them. No editor-switching. No dotfile spelunking. One overlay, everything visible, instant.

The frontend never sees agent-specific formats — every adapter normalizes to one universal skill. Adding the next agent is a single struct entry.

◇ what's inside

Built like a developer tool

One adapter pipeline reads 56 agents. Everything below works on a near-black canvas, keyboard-first, local by default.

Universal scan

One pass across all 56 agents plus the universal AGENTS.md, normalized into a single skill list. Skills, commands, hooks, rules, prompts, workflows.

Live search + facets

Filter by name, description, tag, intent, language, slash command, or hook event. Ctrl+F opens a focused finder; state persists per session.

Tree + grouped views

Card hierarchies for nested skills; grouped lists by agent with brand colors. Skim 300 skills the way you skim a file tree.

Marketplace install PRO

Search public skills on skills.sh and ClawHub, then install with one shell command — copied or run for you, straight from the overlay.

Snapshot + restore PRO

Every update writes a content-hashed snapshot. Diff against the prior version side-by-side, restore in one click — before an agent overwrites your work.

Speaking in Skills PRO

An in-app ReAct agent that doesn't just list your skills — it composes and invokes them together to do real work, right inside the deck.

Hotkey · tray · auto-hide

Rebind the global hotkey from a press-to-capture field. System tray with mode toggles. Auto-hide on focus loss, so it's there only when you call it.

Local + private

A native Rust + Tauri app. Your config never leaves your machine; it talks to GitHub (and, in Pro, the marketplaces) and nothing else.

Themes that fit in

System, Dark, and Light — all with real contrast. Tooltips portal to the body so the overlay's chrome can't clip them.

◇ questions

Good to know

Is Skill Deck free?
Yes. The skill browser — scan, search, views, hotkey, themes — is free forever and MIT-licensed. Pro (a one-time $49) is coming soon and unlocks the in-app agent, marketplace install, GitHub sync, and snapshot history. No subscription.
What's the difference between Free and Pro?
Free finds and shows your skills across every agent. Pro lets you act on them: install from the marketplaces, sync and detect upstream updates, snapshot and restore before overwrites, and run Speaking in Skills — an in-app agent that composes and invokes your skills to do real work.
Which agents does it support?
56 of them, including Claude Code, Cursor, GitHub Copilot, Codex, Windsurf, Gemini CLI, Cline, Roo Code, Continue, Aider, Amazon Q, JetBrains AI, Tabnine, and Augment — plus the universal AGENTS.md format. Adding the next agent is a single entry in registry.rs.
Where does my data go?
Nowhere you didn't ask for. Skill Deck is a native local app. It reads the skill files already on your machine and only talks to GitHub (and, in Pro, the skills.sh / ClawHub marketplaces). Your config stays local.
How do I open the overlay?
A global hotkey — Ctrl+Shift+K on Windows/Linux, ⌘+Shift+K on macOS — from anywhere on your desktop. It's rebindable from a press-to-capture field in settings (modifiers + letters, digits, F-keys, arrows, and more).
Is there a Mac or Linux build?
The prebuilt installer is Windows x64 today. Skill Deck is cross-platform Tauri, so Mac and Linux build from source with one command: pnpm install && pnpm tauri build. Native Mac/Linux installers are on the way.
Is it open source?
Yes — the app is MIT-licensed and the source is on GitHub. Build it yourself anytime, or grab the prebuilt installer below.
The installer shows a security warning?
The binaries aren't code-signed yet, so Windows SmartScreen may say "Windows protected your PC". Click More info → Run anyway. On macOS, right-click the app and choose Open. Signing is planned.
